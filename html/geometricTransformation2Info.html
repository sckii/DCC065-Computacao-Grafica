Geometric Transformation II


When we add an object to another object, the local and world position of this object might be different. In the following example, a sphere was added to a cube. When we move the cube, the local position of the sphere is the same, but its world position changes.

let wp = new THREE.Vector3(); sphere.getWorldPosition( wp );
