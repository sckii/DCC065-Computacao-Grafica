In this example you will learn how to deal with keyboard events in Threejs. Let's see how it works through the example below.

Using the keyboard

One option to work with keyboard in your projects is through the KeyboardState object. Basically you need to (1) create a global variable, (2) during the main loop update the object and (3) check the state of the keys. See the code below: // create global variable var keyboard = new KeyboardState(); (...) // Checking the keys to be used function keyboardUpdate() { keyboard.update(); if ( keyboard.down("left") ) cube.translateX( -1 ); if ( keyboard.down("right") ) cube.translateX( 1 ); } (...) // inside main loop call KeyboardUpdate function function render() { keyboardUpdate(); }
