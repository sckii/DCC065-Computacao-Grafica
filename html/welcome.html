Computer Graphics Course


On this page you will find the basic projects that support the [link:https://sites.google.com/site/computacaograficaufjf/ Computer Graphics course] at [link:https://www2.ufjf.br/ufjf/ UFJF].

All projects were developed in [link:https://threejs.org three.js], a cross-browser JavaScript / API library used to create and display animated 3D graphics on the web. You can access our repository [link:https://github.com/rodrigoluis/CG here].

Contents

You will find in this page projects regarding geometric models, 3D object modeling, scene and virtual camera concepts and topics about lighting and texture usage.

Additionally you will find examples of more advanced techniques such as raytracing, virtual reality and augmented reality.

Our course notes will be available (in portuguese) here.
