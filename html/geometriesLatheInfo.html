Lathe Geometry


You can use this type of geometry to create 'vase like' objects. Basically this object allows the creation of meshes with some sort of axial symmetry. In the project below you can see the generated object and the base points used as the primary curve.

Open in a new tab

Initial curve

In the previous example we generated a set of points that will be used as the base points of our final geometry. These points can be easily generated through some trigonometric functions like sins and cosines. In this project we used the Math.sin function.

var points = []; var numberOfPoints = 12; for (var i = 0; i < numberOfPoints; i++) { points.push(new THREE.Vector2(Math.sin(i*2 / 4.17)+3, i)); }

After the creation of the base points you can create the lathe geometry using, for example, the code below.

// First, create the point vector to be used by the Lathe algorithm var points = generatePoints(); // Set the main properties of the surface var segments = 20; var phiStart = 0; var phiLength = 2 * Math.PI; var latheGeometry = new THREE.LatheGeometry(points, segments, phiStart, phiLength); var object = new THREE.Mesh(latheGeometry, objectMaterial); scene.add(object);
