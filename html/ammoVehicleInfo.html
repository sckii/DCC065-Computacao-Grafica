Ammo - Vehicle


This example uses the ammo.js physics library, that is a direct port of the Bullet physics engine. You can find the basic API documentation of the Bullet engine here.

In this example you can set several parameters of a car, including its mass, engine force, suspension settings etc. This example was based on ones available here.
