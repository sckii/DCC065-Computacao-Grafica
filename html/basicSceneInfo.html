Basic Scene


As we saw in class, we have some basic elements that will appear in almost every project. For example, you will need a scene that will receive all the other elements. You will need also a camera and a renderer. Let's first see our basic project in which we have a cube and interaction through the mouse.

Open in a new tab

Under the hood

Basically we have, at the beginning, the definition of the scene, renderer and camera (the last two created through functions as they will be subject in further classes). To create the camera we must define where it will be created. By default the camera will point to the origin of the world. See the code below to understand how these elements are created.

let scene, renderer, camera, material, light, orbit; // Initial variables scene = new THREE.Scene(); // Create main scene renderer = initRenderer(); // Init a basic renderer camera = initCamera(new THREE.Vector3(0, 15, 30)); // Init camera in this position

The interaction is performed by a orbit control as you can see in the next code.

orbit = new OrbitControls( camera, renderer.domElement ); // Enable mouse rotation, pan, zoom etc.

We will not focus on the creation of the objects for now, but it's worth noting how the objects are added in the scene. In the example below you can see a cube being added to the scene (the material is previously loaded and will be explored in further classes).

let cube = new THREE.Mesh(cubeGeometry, material); scene.add(cube);

Finally, we must create a function in which all the elements will be called in a loop. In this example we create a render function. Inside this function we use two important function of the API - requestAnimationFrame and renderer.render. This will create a loop that causes the renderer to draw the scene every time the screen is refreshed (on a typical screen this means 60 times per second). If you have something to be animated, the variables to be used will be placed inside this render function.

render(); function render() { requestAnimationFrame(render); renderer.render(scene, camera); }
