Linear Interpolation


Linear interpolation (or "Lerp" for short) is a form of interpolation, which involves the generation of new values based on an existing set of values. In principle, a lerp function "eases" the transition between two values over time, using some simple math. This could be used to slide a character between two coordinates, or to animate a change in size.

Threejs has an implementation of lerp in its Vector3 class and can be used in many ways. In the following example, it was used to move an object between its current position and a new one.
