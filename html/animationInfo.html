Animations


Animations in threejs are pretty simple to implement. See the project below in which you can start/stop and change de speed of the animation.

Open in a new tab

Just update de animation variables!

Basically to build simple animations you must update the variables that will govern the movement. In our project we add the rotateCylinder function inside the render loop as you can see in the code below.

function render() { (...) rotateCylinder(); (...) requestAnimationFrame(render); renderer.render(scene, camera) }

Inside the function you must update the variables that you will use to perform the animation. In the code below you can see two of these variable - angle and angle2. They are updated by adding some value in each iteraction of the animation loop (the speed variable is changed in the user interface).

function rotateCylinder() { (...) angle+=speed; angle2+=speed*2; (...) cylinder.matrix.multiply(mat4.makeRotationZ(angle)); // R1 (...) cylinder2.matrix.multiply(mat4.makeRotationY(angle2)); // R1 }
